# Terminal Schortcuts

## Navigation

1. Ctr + A -> Begin
2. Ctr + E -> End
3. Ctr + U -> Clear up line
4. Ctr + d -> Delete Line
5. Ctr + - -> Undo
6. Ctr + L -> Clear Screen
7. Meta + W -> Delete
8. Meta + F -> Word Forward
9. Meta + B -> Word Backwards

## Use of Terminal

* Output Redirection -> >
* Show File contents -> cat
* Append Output -> >>
* Pipe contents -> |
* Search -> Grep + Regx & Search;
